# 👗 Fashion Store Website

A stylish fashion store website designed to showcase clothing and fashion accessories in an organized and visually appealing way. The website includes product categories, shopping banners, upcoming festival sales, discount offers, and payment information to create a complete online shopping experience.

## ✨ Features

- Product Categories Section
  - T-Shirts
  - Sunglasses
  - Shirts
  - Shoes
  - Pants
  - Perfumes

- Shopping Promotion Banner
- Upcoming Sales & Discount Table
- Festival Sale Information
- Payment Options Section
- Clean and User-Friendly Layout
- Simple Navigation Structure

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🎯 Purpose

This project was created to practice frontend web development fundamentals and improve skills in designing structured and user-friendly web pages.

## 📚 Practice Challenge

This project was developed as a solution to the CodeChef HTML practice challenge **"Fashion Store Review"**. The website was built independently for learning purposes while following the requirements of the challenge.
